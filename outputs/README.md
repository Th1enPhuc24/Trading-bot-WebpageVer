## ğŸ“Š Dashboard Types

### 1. Backtest Dashboards (`backtests/`)

**Generated by:** `run_backtest_pipeline.py`, `main_pipeline.py`

**When:** After completing backtest on test data

**Filename format:** `backtest_results_YYYYMMDD_HHMMSS.png`

**Contains:**
- Price chart with all buy/sell signals during backtest
- Equity curve showing account balance progression
- Drawdown visualization
- Training loss history
- Complete trading statistics (win rate, Sharpe ratio, max drawdown, etc.)

**Example:**
```
backtest_results_20251202_084019.png
```

### 2. Live Trading Dashboards (`live/`)

**Generated by:** `trading_bot.py`, `run_with_dashboard.py`

**When:** 
- **Periodic**: Every 12 hours (12 iterations Ã— 60 minutes)
- **Position Closed**: When a trading position is closed (TP/SL/Timeout)
- **Final State**: When bot is stopped (Ctrl+C)

**Filename formats:**
- `live_trading_periodic_YYYYMMDD_HHMMSS.png` - Regular snapshot
- `live_trading_position_closed_YYYYMMDD_HHMMSS.png` - Trade completion
- `live_trading_final_state_YYYYMMDD_HHMMSS.png` - Bot shutdown

**Contains:**
- Real-time price chart with recent signals
- Neural network output values
- Current equity curve
- Active position information
- Live performance metrics
- Recent trade history

**Example:**

## ğŸ—‚ï¸ File Naming Convention

**Timestamp format:** `YYYYMMDD_HHMMSS`
- `YYYY` - Year (e.g., 2025)
- `MM` - Month (01-12)
- `DD` - Day (01-31)
- `HH` - Hour (00-23)
- `MM` - Minute (00-59)
- `SS` - Second (00-59)

**Example:** `20251202_143055` = December 2, 2025 at 14:30:55
